Shared integration framework for all VentorTech e-commerce connectors.

* The screenshot shows all four supported platforms; this core module provides the common logic for individual connectors.
Important: This core module does not include any connector logic by itself. You must purchase and install a corresponding VentorTech connector module (e.g. integration_shopify) for the integration to function. Links to each connector are provided below.

The Odoo E-commerce Connector Core provides the shared framework powering all VentorTech connectors. When paired with your chosen connector module, you unlock:

  • Order import/export with status updates & automated workflows
  • Two-way products synchronization (attributes, images, custom fields)
  • Real-time or scheduled inventory control across warehouses
  • Customer data sync & intelligent deduplication/mapping
  • Pricing & discount synchronization
  • Webhook support for instant updates and centralized error handling
  • Performance optimizations, logging & unified maintenance
